## ps

```
$ ps             # 列出当前shell里当前用户的进程

$ ps -u chris    # 列出chris运行的所有进程
$ ps -fu chris   # 如上，含ppid

$ ps -e          # 显示运行的所有进程
$ ps -el         # 详细列表
$ ps -ef         # 完整格式列表

$ ps ax          # 显示运行的所有进程，简短的BSD风格
$ ps aux         # 显示运行的所有进程，详细的BSD风格
$ ps auwx        # 宽格式
$ ps auwwx       # 宽度不限
```
