收集开发过程中的一些技巧

## 三元表达式

在javascript等语言中有三元表达式，用起来比较方便，elixir中没有对应的语法。

### javascript

```js
const v = a == 1 ? 'hello' : 'world';
```

在 Elixir中可以这样

```elixir
v = if a == 1, do: "hello", else: "world"
```

或者可以使用 `&&` 和 `||`

```elixir
v = a == 1 && "hello" || "world"
```

看起来上面的语句更接近也更简洁：）
